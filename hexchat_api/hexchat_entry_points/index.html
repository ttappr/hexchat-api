[−][src]Module hexchat_api::hexchat_entry_points

This module holds the DLL entry points for this plugin library. These exported functions are what Hexchat links to directly when the plugin is loaded.

These functions register the plugin info with Hexchat and set up to "catch" any panics that Rust-side callbacks might "raise" during execution. Panics are displayed in the currently active Hexchat window/context. The debug build of the library will include a stack trace in the error message.
